4.5. 更换关节

4.5.1. 精度说明

机器人的精度，分为重复定位精度和绝对定位精度：

  • 对于只对重复定位精度有要求的场景（如搬运、码垛、点胶、点焊），这些场景只要每次都能运行到同一个固定的点位就可以了，可以更换关节。

  • 对绝对定位精度要求较高的场景，不建议自行更换关节，更换完关节后，存在绝对定位精度丢失、轨迹精度降低、无法满足工艺要求的情况；更换关节后，必须使用激光跟踪仪标定，才可恢复绝对定位精度。要求绝对定位精度的场景，主要为以下场景：

    • 离线编程场景（导入机械模型，自动生成示教轨迹）。

    • 视觉（2D、3D）用于视觉引导、纠偏等场景。

    • 存在具体的工艺要求，如直线涂胶、焊接，长距离直线穿孔；圆弧涂胶、焊接等。

4.5.2. 关节螺栓紧固

4.5.2.1. 关节螺栓拧紧顺序

1. 螺栓标号

  • 关节螺栓：输出侧朝右，以水平轴线为基准，水平轴线右下侧标为1号螺栓，顺时针标记全部螺栓。

  • 静止关节小端螺栓（分体式小臂）：大端朝向右侧，以水平轴线右下角为1号螺栓，顺时针标记全部螺栓。

  • 静止关节或一体式小臂大端螺栓：小端朝向右侧，以水平轴线右下角为1号螺栓，顺时针标记全部螺栓。

2. 螺栓拧紧

  • 螺栓总数为12时，首先按照1-7-6-12-3-9-10-4-2-8-11-5顺序预锁紧至目标力矩的50%，再按照同样顺序锁紧至目标力矩。

  • 螺栓总数为15时,首先按照1-9-7-15-3-11-13-5-2-10-14-6-4-12-8顺序预锁紧至目标力矩的50%，再按照同样顺序锁紧至目标力矩。

备注

  • 输入接口所连接的线束，是指从上一关节（或管臂） 到待更换关节之间的线束。

  • 输出接口所连接的线束，是指从待更换关节到下一关节（或管臂） 之间的线束。

  • 示例（以二关节为待更换关节）：上一关节为一关节，下一关节为三关节。

4.5.4. Zu15更换关节

Zu15的一关节至三关节的固定螺栓为外六角螺栓，采用轴向螺钉固定的方式；四关节至六关节的固定螺栓为外六角螺栓，采用径向螺钉固定的方式，拆装方式略有差异，因此分开说明。

关节拆卸：

  • 拆卸前准备

    • 断开电源，并佩戴防静电手环。

    • 移动机器人至一个便于拆卸的位置，如有必要，从工作单元上拆下整个机器人，并将其放在固定工作台面上。

  • 拆卸装饰环

    • 使用PH1十字螺丝刀旋松4颗一、二关节之间的装饰环固定的M3十字螺钉，取下两个装饰环。

    • 使用PH1十字螺丝刀旋松2颗大臂、二关节之间的装饰环固定的M3十字螺钉，取下两个装饰环。

  • 分离大臂与二关节

    • 使用2.5mm内六角螺丝刀旋松3颗固定闷盖的M3内六角螺钉，拆下二关节的闷盖。

    • 使用斜口钳剪断固定EtherCAT线、红色和黑色电源线的扎带。使用镊子将EtherCAT线、红色和黑色电源线的端子拔下。

    • 使用开口扳手旋松并取下固定关节的外六角螺栓和垫片。扳手和螺栓的规格参考 关节螺栓规格 。

    • 缓慢将大臂取下，同时将线束从二关节的出线孔逐一抽出，为防止线束端子卡住，推荐先抽出电源线，再抽出EtherCAT线。

  • 分离一关节与二关节

    • 使用2.5mm内六角螺丝刀旋松4颗固定环形灯壳的M3内六角螺钉。

    • 使用镊子拔下环形灯壳内灯板上的环形灯线端子，取下一关节的环形灯壳。

    • 使用斜口钳剪断固定EtherCAT线、红色和黑色电源线的扎带。

    • 使用镊子将EtherCAT线、红色和黑色电源线的端子拔下，此时只需拔下连接至二关节的线束端子，即连接至 驱动板接口定义 中输出接口的线束端子。

    • 使用开口扳手旋松并取下固定关节的外六角螺栓和垫片。扳手和螺栓的规格参考 关节螺栓规格 。

    • 缓慢将二关节取下，同时将线束从一关节的出线孔逐一抽出，为防止线束端子卡住，推荐先抽出电源线，再抽出EtherCAT线。

  • 取出二关节线束

    • 使用2.5mm内六角螺丝刀旋松1颗固定扎线固定座的M3内六角螺钉，取下扎线固定座。

    • 依次将线圈固定件和线圈固定钣金穿过关节线束并取下。

    • 使用2.5mm内六角螺丝刀旋松3颗固定法兰固线钣金的M3内六角螺钉，将线束从二关节法兰中心孔抽出。

关节安装：

  • 穿入二关节线束

    • 将密封油脂装入针筒内，并将油脂注入新关节法兰面的凹槽内，确保密封油脂涂抹均匀且充满凹槽。

    • 取对应O型圈，将其放入二关节法兰面凹槽中，按压O型圈，使其完全嵌入到凹槽中。

    • 将拆下的线束从二关节法兰面的中心孔侧穿过。

    • 使用2.5mm内六角螺丝刀拧紧3颗固定法兰固线钣金的M3内六角螺钉。

    • 将线圈固定件套在线束热缩管上，并将线圈固定钣金置于散热器上的钣金安装位置上。

    • 将1颗M3内六角螺钉穿过扎线固定座，并将其置于线圈固定钣金上，使用2.5mm内六角螺丝刀拧紧这颗螺钉。

    • 使用扎带将线圈固定件上方的热缩管扎上，使用斜口钳将多余扎带剪掉，注意剪扎带时切勿剪到线束。

  • 组装一关节与二关节

    • 将二关节的线束穿过一关节的出线孔，为防止线束端子卡住，推荐先穿过EtherCAT线，再穿过电源线。

    • 将二关节法兰面的凹槽对准一关节的凸台，使二关节法兰面与一关节贴合。

    • 将固定关节的外六角螺栓和垫片放入一关节的螺纹孔中，并使用开口扳手按对角顺序拧紧螺栓。螺栓拧紧顺序参考 关节螺栓紧固 ，扳手和螺栓的规格参考 关节螺栓规格 。

  • 组装大臂与二关节

    • 将密封油脂装入针筒内，并将油脂注入大臂法兰面的凹槽内，确保密封油脂涂抹均匀且充满凹槽。

    • 取对应O型圈，将其放入大臂法兰面凹槽中，按压O型圈，使其完全嵌入到凹槽中。

    • 将大臂的线束穿过二关节的出线孔，为防止线束端子卡住，推荐先穿过EtherCAT线，再穿过电源线。

    • 将大臂法兰面凹槽对准二关节的凸台，使大臂法兰面与二关节贴合。

    • 将固定关节的外六角螺栓和垫片放入二关节的螺纹孔中，并使用开口扳手按对角顺序拧紧螺栓。螺栓拧紧顺序参考 关节螺栓紧固 ，扳手和螺栓的规格参考 关节螺栓规格 。

  • 线束连接与整理

    • 在一关节和二关节闷盖侧，将EtherCAT线、红色电源线、黑色电源线连接至一关节和二关节驱动板上对应的接口。接口说明参考 驱动板接口定义 。

    • 整理线束并使用扎带将线束成组固定，使用斜口钳将多余扎带齐根剪掉，注意剪扎带时切勿剪到线束。

  • 安装闷盖和环形灯壳

    • 将密封油脂装入针筒内，在闷盖边缘的凹槽里涂抹一圈密封油脂，确保密封油脂涂抹均匀且充满凹槽。

    • 取对应O型圈，将其放入闷盖凹槽中，按压O型圈，使其完全嵌入到凹槽中。

    • 将闷盖置于二关节外壳上，使闷盖上的孔对准外壳上的螺纹孔，使用2.5mm内六角螺丝刀拧紧3颗M3内六角螺钉，以固定二关节闷盖。安装闷盖时切勿压到驱动版上的线束。

    • 在环形灯壳边缘的凹槽里涂抹一圈密封油脂，确保密封油脂涂抹均匀且充满凹槽。

    • 取对应O型圈，将其放入环形灯壳凹槽中，按压O型圈，使其完全嵌入到凹槽中。

    • 将关节驱动版上的环形灯线插到环形灯壳的灯板上。

    • 将环形灯壳置于一关节外壳上，使环形灯壳上的孔对准二关节外壳上的螺纹孔，使用2.5mm内六角螺丝刀拧紧4颗M3内六角螺钉，以固定一关节环形灯壳。安装环形灯壳时切勿压到驱动版上的线束。

  • 安装装饰环

    • 将一个装饰环放入一、二关节之间，使其拼接处与关节标定刻线对齐。

    • 使用PH1十字螺丝刀拧紧2颗M3十字螺钉，固定装饰环。

    • 将另一个装饰环放入一、二关节之间，使两个装饰环的卡扣对齐。

    • 使用PH1十字螺丝刀拧紧2颗M3十字螺钉，固定装饰环。

    • 将一个装饰环放入大臂、二关节之间，使其拼接处与关节标定刻线对齐。

    • 使用PH1十字螺丝刀拧紧1颗M3十字螺钉，固定装饰环。

    • 将另一个装饰环放入大臂、二关节之间，使两个装饰环的卡扣对齐。

    • 使用PH1十字螺丝刀拧紧1颗M3十字螺钉，固定装饰环。

    • 缓慢将四关节取下，同时将线束从四关节的出线孔逐一抽出，为防止线束端子卡住，推荐先抽出电源线，再抽出EtherCAT线。

  • 取出四关节线束

    • 使用2.5mm内六角螺丝刀旋松并取下1颗固定线圈固定钣金的M3内六角螺钉。

    • 将线圈固定件连同关节线束从关节驱动板侧取出。

关节安装：

  • 穿入四关节线束

    • 将密封油脂装入针筒内，并将油脂注入新关节法兰面的凹槽内，确保密封油脂涂抹均匀且充满凹槽。

    • 取对应O型圈，将其放入四关节法兰面凹槽中，按压O型圈，使其完全嵌入到凹槽中。

    • 将拆下的线束从四关节驱动板侧的中心孔侧穿过。

    • 将线圈固定钣金置于螺柱上，使用2.5mm内六角螺丝刀拧紧1颗固定线圈固定钣金的M3内六角螺钉。

    • 使用扎带将线圈固定件上方的热缩管扎上，使用斜口钳将多余扎带剪掉，注意剪扎带时切勿剪到线束。

  • 组装小臂与四关节

    • 将密封油脂装入针筒内，并将油脂注入小臂法兰面的凹槽内，确保密封油脂涂抹均匀且充满凹槽。

    • 取对应O型圈，将其放入小臂法兰面凹槽中，按压O型圈，使其完全嵌入到凹槽中。

    • 将小臂的线束穿过四关节的出线孔，为防止线束端子卡住，推荐先穿过EtherCAT线，再穿过电源线。

    • 将小臂法兰面凹槽对准四关节的凸台，使小臂法兰面与四关节贴合。

    • 将径向螺钉定位套放入四关节侧面的4个锥孔内。

    • 将4颗较长的螺栓放入定位套内，使用内六角螺丝刀按对角顺序拧紧4颗螺栓。螺栓的规格参考 关节螺栓规格 。

    • 将剩余的8颗较短的螺栓放入四关节的螺纹孔中，并使用内六角螺丝刀按对角顺序拧紧螺栓。螺栓拧紧顺序参考 关节螺栓紧固 ，螺栓的规格参考 关节螺栓规格 。

  • 组装四关节与五关节

    • 将四关节的线束穿过五关节的出线孔，为防止线束端子卡住，推荐先穿过EtherCAT线，再穿过电源线。

    • 将四关节法兰面凹槽对准五关节的凸台，使四关节法兰面与五关节贴合。

    • 将径向螺钉定位套放入五关节侧面的4个锥孔内。

    • 将4颗较长的螺栓放入定位套内，使用内六角螺丝刀按对角顺序拧紧4颗螺栓。螺栓的规格参考:ref:关节螺栓规格 。

    • 将剩余的8颗较短的螺栓放入五关节的螺纹孔中，并使用内六角螺丝刀按对角顺序拧紧螺栓。螺栓拧紧顺序参考 关节螺栓紧固 ，螺栓的规格参考 关节螺栓规格 。

  • 线束连接与整理

    • 在四关节和五关节闷盖侧，将EtherCAT线、红色电源线、黑色电源线连接至四关节和五关节驱动板上对应的接口。接口说明参考 驱动板接口定义 。

    • 整理线束并使用扎带将线束成组固定，使用斜口钳将多余扎带齐根剪掉，注意剪扎带时切勿剪到线束。

  • 安装闷盖

    • 将密封油脂装入针筒内，分别在四关节和五关节闷盖边缘的凹槽里涂抹一圈密封油脂，确保密封油脂涂抹均匀且充满凹槽。

    • 取对应O型圈，将其放入闷盖凹槽中，按压O型圈，使其完全嵌入到凹槽中。

    • 将一个闷盖置于四关节外壳上，使闷盖上的孔对准关节外壳上的螺纹孔，使用2.5mm内六角螺丝刀拧紧3颗M3内六角螺钉，以固定四关节闷盖。安装闷盖时切勿压到驱动版上的线束。

    • 将另一个闷盖置于五关节外壳上，使闷盖上的孔对准关节外壳上的螺纹孔，使用2.5mm内六角螺丝刀拧紧3颗M3内六角螺钉，以固定五关节闷盖。安装闷盖时切勿压到驱动版上的线束。

  • 安装装饰环

    • 将一个装饰环放入小臂、四关节之间，使其拼接处与关节标定刻线对齐。

    • 使用PH1十字螺丝刀拧紧2颗M3十字螺钉，固定装饰环。

    • 将另一个装饰环放入小臂、四关节之间，使两个装饰环的卡扣对齐。

    • 使用PH1十字螺丝刀拧紧2颗M3十字螺钉，固定装饰环。

    • 将一个装饰环放入四、五关节之间，使其拼接处与关节标定刻线对齐。

    • 使用PH1十字螺丝刀拧紧2颗M3十字螺钉，固定装饰环。

    • 将另一个装饰环放入四、五关节之间，使两个装饰环的卡扣对齐。

    • 使用PH1十字螺丝刀拧紧2颗M3十字螺钉，固定装饰环。

关节拆卸：

    • 断开电源，并佩戴防静电手环。

    • 使用吊起设备将机器人移动至一个便于拆卸的位置，如有必要，从工作单元上拆下整个机器人，并将其放在固定工作台面上。

    • 使用PH1十字螺丝刀旋松4颗一、二关节之间的装饰环固定的M3十字螺钉，取下两个装饰环。

    • 使用斜口钳剪断固定EtherCAT线、红色和黑色电源线的扎带。使用镊子将EtherCAT线、红色和黑色电源线的端子拔下。

    • 使用2.5mm内六角螺丝刀旋松6颗固定环形灯壳的M3内六角螺钉。

    • 使用镊子拔下环形灯壳内灯板上的环形灯线端子，取下一关节的环形灯壳。

    • 使用斜口钳剪断固定EtherCAT线、红色和黑色电源线的扎带。

    • 使用镊子将EtherCAT线、红色和黑色电源线的端子拔下，此时只需拔下连接至二关节的线束端子，即连接至 驱动板接口定义 中输出接口的线束端子。

    • 使用开口扳手旋松并取下固定关节的外六角螺栓和垫片。扳手和螺栓的规格参考 关节螺栓规格 。

    • 缓慢将二关节取下，同时将线束从一关节的出线孔逐一抽出，为防止线束端子卡住，推荐先抽出电源线，再抽出EtherCAT线。

4.5.6. 软件操作

更换关节后，为确保关节正常运行，须通过软件依次执行以下四项操作：

  • 关节序列号写入

  • 关节参数写入

  • 升级伺服

  • 关节校准

关节序列号写入：

  1. 连接机器人连接线至控制柜，使用手柄或示教器使控制柜上电。

  2. 打开Coboπ软件，连接机器人，进入 主页，点击 电源 和 使能 按钮，使机器人上电并使能。

  3. 进入 首页，点击 按钮，再点击 维护 按钮，输入密码（JAKAAMAZING），进入 维护 界面。

  4. 进入 伺服调试 界面，点击 进入调试 开始调试。

  5. 打开伺服调试软件（可联系JAKA技术服务工程师获取），进入 驱动器调试 界面，在 机器人IP 输入框内输入当前机器人的IP地址，点击 连接。

    备注

    机器人IP可在Coboπ软件顶部导航栏查看。

  6. 进入 生产 界面，选择更换的关节，点击 读取参数，在 9009序列号 后的输入框内输入更换关节的序列号，点击 设置。

关节参数写入：

  1. 进入 驱动器调试 界面，在 编号 列选择 14 总线设置 ，将 1401 本机节点号 的值修改为更换的关节号，点击 写入选中 按钮写入参数。

  2. 在 编号 列选择 22 编码器参数 ，将 2201 ABS编码器单圈分辨率 和 2219 总线指令分辨率 的值修改为 23 ，点击 写入选中 按钮写入参数。

升级伺服：

  1. 回到Coboπ软件，进入 主页，点击 使能 和 电源 按钮，使机器人下使能、下电。

  2. 点击 设置 > 版本升级 进入 版本升级 界面。

  3. 点击 选择文件，找到已下载至本地的伺服升级包，点击 打开，上传伺服升级包（升级包可联系JAKA技术服务人员获取，文件后缀为 .tar.gz ），点击 开始升级 。

    重要

    升级包的名称不可修改，必须保持原文件名。

  4. 升级完成后，系统会提示重新登录，登录后可直接上电。

关节校准：

  1. 打开Coboπ软件，进入 主页，点击 电源 和 使能 按钮，使机器人上电并使能。

  2. 进入 手动 界面，点击虚拟摇杆，使机器人各关节的标定刻线对齐，此时机器人处于校准姿态。标定刻线位置和姿态如下。

    • 标定刻线位置：

    若更换的关节在手动操作出现报错“超过软限位”，则按照步骤5先将该关节校准一次，再回到 手动 界面，继续手动调整至校准姿态。

  3. 机器人移至关节校准姿态后，返回 首页，点击 按钮，再点击 维护 按钮，输入密码（JAKAAMAZING），进入 维护 界面。

  4. 进入 关节校准 界面，点击 关节x校准 （x为更换的关节号），此时控制器将此关节当前角度作为零位角度。
